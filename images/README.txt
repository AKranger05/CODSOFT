IMAGE PLACEMENT GUIDE
=====================

Add your images to this folder with these exact filenames:

HERO SECTION:
- hero-fashion.jpg (600x800px recommended)
  Main hero image showcasing your brand

COLLECTION SECTION:
- mens-collection.jpg (400x500px recommended)
  Men's clothing collection image
- womens-collection.jpg (400x500px recommended)  
  Women's clothing collection image
- accessories.jpg (400x500px recommended)
  Accessories collection image

ABOUT SECTION:
- about-story.jpg (500x600px recommended)
  Image representing your brand story/team

IMAGE TIPS:
- Use high-quality images
- Ensure good lighting
- Keep consistent style across all images
- Optimize file sizes for web (under 500KB each)
- Use JPG format for photos, PNG for graphics with transparency

The placeholder boxes will automatically hide when you add the images!
